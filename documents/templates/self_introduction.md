# 自己紹介スクリプト（カジュアル面談用）

カジュアル面談・選考面接の冒頭で使う自己紹介の原稿。

冒頭で「簡単に自己紹介をお願いします」と聞かれた際に、迷わず話せるようテンプレート化したもの。状況に応じて 1分30秒版 / 45秒版 を使い分ける。

## 1分30秒版（標準）

```text
本日はお時間をいただきありがとうございます、大川尚斗と申します。

現在は株式会社アトムズテクノロジーで、フロントエンドエンジニアとして
Next.js / TypeScript を中心とした開発に従事しています。

エンジニアとしてのキャリアは2021年からで、現在で約5年になります。
最初はWordPressでのコーポレートサイト制作からスタートし、
現職ではShopify PlusでのEC開発、Next.jsでの新規プロダクト開発、
Laravelでの API実装まで幅広く担当してきました。

直近では4名のメンバーをマネジメントするリーダーポジションも経験し、
顧客折衝・技術選定・コードレビューまで一貫して担当しています。

技術面の強みは、Figmaでのデザインから Next.js での実装、
本番デプロイまでを一貫して担当できることです。
個人でもFigmaを使ってポートフォリオサイトを設計・実装しており、
デザインとエンジニアリングを横断できる点が自分の強みだと考えています。

今後はUI/UXを起点に価値を生み出せるエンジニアを目指しており、
本日はそういった環境について、お話を伺えればと思っております。
よろしくお願いいたします。
```

### 構成解説

| パート | 内容 | 目安秒数 |
| --- | --- | --- |
| ① 挨拶 | 名前と感謝 | 5秒 |
| ② 現職 | 何をしているか | 15秒 |
| ③ キャリア概要 | これまでの経験の幅 | 25秒 |
| ④ 強み | 一番のアピール | 25秒 |
| ⑤ 締め | 今後と面談への期待 | 15秒 |

## 45秒版（短縮）

「短くお願いします」と言われた時用の予備バージョン。

```text
大川尚斗と申します。本日はよろしくお願いいたします。

2021年からフロントエンドエンジニアとしてキャリアをスタートし、
現在で約5年になります。
直近ではNext.js / TypeScriptを中心に、
ECサイトや新規プロダクトの開発を担当しています。

強みはFigmaでのデザインから実装まで一貫して担当できることで、
個人でもポートフォリオサイトを設計から構築しています。

本日はお話を伺えるのを楽しみにしておりました。
よろしくお願いいたします。
```

## 話し方のコツ

### やること

- 棒読みしない（自然に話す）
- 間を取る（早口にならない）
- 目線をカメラに（画面の人物ではなくレンズ）
- 笑顔を意識（リモート面談は表情が伝わりにくい）

### やらないこと

- 暗記して呪文のように話す
- 1分以上かけて話す（相手が飽きる）
- 専門用語を多用する（採用担当者がエンジニアじゃない場合もある）

## カスタマイズポイント

企業ごとに最終行のみ差し替える。

| デフォルト | カスタム例 |
| --- | --- |
| 本日はそういった環境について、お話を伺えればと思っております | 本日は御社の〇〇な点に興味を持っておりまして、お話伺えればと思っております |
