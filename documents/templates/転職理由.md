# 退職理由の言い換え準備

面接で「退職理由」を聞かれたときの回答テンプレート。
**本音をそのまま話さず、前向きな表現に変換して話す**ことが基本方針。

---

## 📌 前提：退職の経緯（整理）

| 区分 | 内容 |
| ---- | ---- |
| 前職（株式会社80） | 会社都合による新会社設立に伴い移籍 → 現職へ |
| 現職（株式会社アトムズテクノロジー） | 受託開発中心。転職活動中 |
| 転職理由の核心 | 自社プロダクトで「デザイン × 実装」を長期視点で深められる環境へ |

## ✅ メインの回答（面接で使う）

### 結論から入るパターン（推奨）

```text
一言で言うと、「デザインとエンジニアリングを一貫して深められる環境」に
挑戦したいというのが転職の理由です。

現職は受託開発が中心で、技術力や対応範囲という意味では非常に鍛えられました。
ただ、受託という性質上「納品して終わり」になりやすく、
デザインの意図や運用後の改善まで継続的に関われる機会が限られています。

直近では自社コーポレートサイトをUI設計から実装・運用フロー整備まで
一人称で完結させた経験があり、この「設計〜運用の一貫した関わり方」を
自社プロダクトの規模で継続的に実践できる環境を求めて転職活動をしています。
```

### 深掘りされた場合の補足

```text
現職でもデザイン領域に踏み込む努力はしており、Figmaを活用した
コンポーネント設計やデザイナーとの協業経験も積んできました。
ただ、受託主体の環境ではどうしても短期の納品サイクルになりやすく、
事業会社でプロダクトを長期視点で磨き続ける経験が積みにくいと感じています。
御社のような自社開発の環境で、そのギャップを埋めていきたいと考えています。
```

## 📋 本音 → 面接向け変換表

| 本音（言わない） | 面接向けの言い換え |
| ---------------- | ------------------ |
| 受託がしんどい、評価されない | 自社プロダクトで長期視点の開発に携わりたい |
| 給料を上げたい | より専門性を活かせる環境で、相応の評価を受けたい |
| 会社の将来性が不安 | 自社サービスを持つ安定した事業基盤の会社で働きたい |
| 裁量がない・決めさせてもらえない | 技術選定・設計判断に主体的に関われる環境を求めている |
| 前職が会社都合で移籍になった | 会社都合による新会社設立に伴い現職へ移籍（事実として簡潔に） |

## 🏢 会社別カスタマイズ

### 株式会社 Scene Live（カジュアル面談 → 選考進行中）

Scene Liveはエンタメ・ライブ領域の自社プロダクトを持つ会社のため、
「プロダクトへの長期貢献」と「ユーザー体験への関心」を強調する。

```text
御社のように、ユーザーの体験そのものをプロダクトとして持つ環境で
働きたいという気持ちが大きいです。

受託開発では、エンドユーザーの声が届きにくい構造があります。
御社のプロダクトのように、ユーザーが実際に使い続けてくれるものを
フロントエンドとデザインの両面から磨き続ける仕事に
強く惹かれています。
```

### 汎用（自社開発企業向け共通テンプレ）

```text
現職の受託開発で技術の幅は広げてきましたが、
自社プロダクトをユーザー視点で継続的に改善していく経験を積みたく、
事業会社への転職を決めました。
特に御社のような、デザインとエンジニアリングが
近い距離で動ける環境を探していました。
```

## ⚠️ 絶対に言わないこと

- 「待遇・給与への不満」（→ お金目当てに見える）
- 「上司・同僚との人間関係トラブル」（→ 協調性を疑われる）
- 「会社の将来が不安だった」（→ ネガティブな印象）
- 「受託がきつくて嫌になった」（→ 逃げの転職に見える）
- 前職（株式会社80）の詳細な事情（→ 聞かれたら「会社都合の移籍」とだけ答える）

## ❓ 深掘り質問と回答パターン

### 「現職で解決できないんですか？」

```text
できる限りの努力はしています。直近の自社コーポレートサイトでは
UI設計から実装・運用整備まで一貫して担当しました。
ただ、受託主体という構造上、継続的なプロダクト改善の機会が
限られるのは変わらない課題です。
転職というよりは「より自分の強みを活かせる場所への移動」と捉えています。
```

### 「前の会社はなぜ辞めたんですか？（株式会社80について）」

```text
会社都合による新会社設立に伴い移籍したため、転職活動はしていませんでした。
今回が初めての自主的なキャリア転換になります。
```

### 「転職活動はどのくらい進んでいますか？」

```text
いくつかの企業様とカジュアル面談をさせていただいており、
御社が最も自分の志向と合っていると感じています。
（→ 他社の状況は正直に話しすぎず、「御社が最有力」のニュアンスを出す）
```

### 「いつ頃の入社を考えていますか？」

```text
現職の引き継ぎ等を考慮すると、2026年8〜10月頃を目処にしています。
御社のご都合に合わせて相談させていただきたいと思っています。
```

## 💡 話し方のコツ

- **前向きな理由を先に言う**（ネガティブな理由から入らない）
- **「〜したい」「〜を目指したい」** という未来志向の言葉を使う
- 現職への不満を聞かれても、**「鍛えられた・感謝している」** を必ず一言入れる
- 短く話す（1分以内）。深掘りされたら補足する構造にする
